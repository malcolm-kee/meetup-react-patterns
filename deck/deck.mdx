# Render Props

```jsx
<BehaviorComponent
    option={someOptionValue}
    render={data => 
        <VisualComponent data={data} />}
/>
```
---
# Render Props = 
# delegates rendering

```jsx
<Toggle
    // delegate behavior
    onToggle={toggleHandler}    

    // delegate rendering
    render={({ toggle, on }) =>
        <Switch on={on} onClick={toggle} />} 
/>
```

---
# Alternative Format
```jsx
<BehaviorComponent>
    {data => <VisualComponent data={data} />}
</BehaviorComponent>
```
